🤖 Discord Automation Core
Este repositório contém o núcleo de um bot de automação para Discord desenvolvido em Python. O projeto foca em fornecer uma base sólida e escalável para gestão de comunidades e automação de tarefas, servindo como o primeiro estágio para uma futura integração com agentes inteligentes.

🎓 Contexto do Projeto
Desenvolvido como parte do meu portfólio técnico em Engenharia de Computação pela UFC, este bot aplica conceitos fundamentais de:

Programação Assíncrona: Utilização intensiva de asyncio para garantir alta performance e baixa latência.

Persistência de Dados: Gestão de informações através de banco de dados local (SQLite) para armazenamento de logs e interações.

Arquitetura Modular: Estrutura preparada para expansão de funcionalidades sem comprometer a estabilidade do sistema.

🛠️ Tecnologias Utilizadas
Linguagem: Python 3.x.

Framework Principal: discord.py (ou disnake).

Banco de Dados: SQLite (arquivo criticas).

Hospedagem: Preparado para deploy via Discloud.

🚀 Funcionalidades Atuais
Gestão de Críticas/Logs: Sistema automatizado de recebimento e armazenamento de dados via banco de dados persistente.

Comandos Administrativos: Lógica de moderação e automação de rotinas de servidor.

Configuração de Deploy: Arquivos de ambiente prontos para integração contínua e hospedagem em nuvem.

📈 Roadmap de Evolução
Como profissional que atua com programação de agentes de IA, as próximas etapas deste projeto incluem:

Refatoração para Cogs: Divisão modular da lógica para facilitar a manutenção.

Integração com LLMs: Implementação de agentes baseados em GPT para suporte e moderação inteligente.

Dashboard Web: Interface em ReactJS para controle administrativo do bot em tempo real.
